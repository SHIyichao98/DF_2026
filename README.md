# Surrogate Modeling Workshop - Preparation Package

Start with the guide in your preferred language:

- [English setup guide](guides/Workshop_Setup_Guide_EN.md)
- [中文准备指南](guides/Workshop_Setup_Guide_ZH.md)

Support files:

- `notebooks/Environment_Check.ipynb` - one-click readiness check; it does not train a machine-learning model
- `environment/environment.yml` - recommended Python 3.11 Conda environment
- `environment/requirements.txt` - equivalent pip dependencies
- `environment/README.md` - short local-install commands

The workshop datasets and teaching notebooks will be distributed separately.
